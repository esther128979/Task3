# מדריך התקנה - CloudSnap

ברוכה הבאה! כאן תלמדי איך להפעיל את CloudSnap תוך פחות מ-2 דקות.

## דרישות קדם
* מערכת הפעלה Windows 10+ או macOS.
* מיקרופון תקין.

## שלבי התקנה

1. **הורדת הקובץ:**
   הורידי את המערכת מהאתר הרשמי (לינק דמיוני).

2. **הרצה בטרמינל:**
   פתחי את שורת הפקודה והקלידי:
   ```bash
   npm install -g cloudsnap-cli
